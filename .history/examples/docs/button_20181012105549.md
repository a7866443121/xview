# Button 按钮
----
### 基础用法
使用```type```、```radius```和```round```属性来定义 Button 的样式。

### Attributes
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|---------- |-------- |---------- |-------------  |-------- |
| type     | 类型   | string    | plain , primary , success , info , warning , danger |     —    |
| round     | 圆端按钮  | Boolean    | — | false   |
| radius  | 圆角按钮    | boolean   | —   | false   |
| disabled  | 是否禁用状态    | boolean   | —   | false   |
